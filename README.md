# C++ Sleep Library

This C++ library allows you to wait the time you choose (like in python time.sleep(3) but in seconds)

## Usage

1. Download
2. Drag sleep.h in your folder

```cpp
#include "sleep.h"

int main() {
    // Your code here

    // Sleep for 3 seconds
    Sleep::sleep(3);

    // Rest of your code
    return 0;
}

```
